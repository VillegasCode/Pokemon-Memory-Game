## :dart: About ##

This is a memory game created with PokÃ©mon characters, you have to find the pair so that the tiles match, when they match, the tiles remain static and appear showing their faces.

The statistics show us "Aciertos - Successes", "Movimientos - Moves" and "Tiempo - Time" and when the game ends they show us a button called "Volver a Jugar - Replay" to play again.
